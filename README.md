# 优医问诊APP

#### 介绍
优医问诊是连接医生和患者的移动医疗行业APP，通过高效、快速的在线问诊服务，为患者提供健康帮助，涵盖了找医生、快速问诊、开药门诊、疾病查询，健康百科等功能，本项目是优医问诊的开发笔记。

#### 访问地址

- https://fhguo.github.io/yywz-doc/

#### 安装教程

> 提前安装运行环境[nodejs](https://nodejs.org/en)，版本号>=18

1. 安装

   ```bash
   # or pnpm or yarn
   npm install
   ```

2. 运行

   ```bash
   npm run dev
   ```

3. 打包

   ```bash
   npm run docs:build
   ```
